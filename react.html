REACT NATIVE

React Native combines the best parts of native development with React, a best-in-class JavaScript library for building user interfaces.

Use a little—or a lot. You can use React Native today in your existing Android and iOS projects or you can create a whole new app from scratch.

First lets talk about "import". import is just like we declare variables in Java-Script. import is used to declare the components of react-native. If you don't write the component in import, you will not be able to see your output. Other important feature is render which is used to return the code and display anything on the screen. Code written in return() is in JSX Format.

We use two curly brackets in JSX. Now the question arises - WHY ? I have the answer for you. One bracket is for writing the JavaScript code and other bracket is for JSON style.

So I hope you are clear with React Native till here. Let's learn about the pre-defined components of react native. Firstly, "View" is the empty container which holds the styling property of any tag. Like, we did in our class, we cannot style the button tag, so we used the view tag to style our button tag. Text is the another tag to write the word, phrase or a sentence on the screen. In button tag even we can give color and title.

Last but not least, I am going to tell you that App is the main class which extends the parent class i.e; Component class. One more interesting thing - You can even create your own react native by declaring the class and extending the component class.
